# Projeto Android

Projeto desenvolvido com **HTML5** e **CSS3**, inspirado em blogs e sites de artigos. O objetivo principal foi praticar conceitos fundamentais de front-end, estruturação semântica de páginas, estilização com CSS e responsividade.

**Acesse o site:**  
https://dev-emanuelpereira.github.io/projeto-android/

---

## Visão Geral

Este projeto apresenta uma página informativa sobre o Android, utilizando um layout limpo, organizado e responsivo. Ele foi construído sem o uso de frameworks ou bibliotecas externas, focando exclusivamente em HTML e CSS puro.

---

## Tecnologias Utilizadas

- **HTML5**
  - Estrutura semântica
  - Uso de tags como `header`, `main`, `section`, `article` e `footer`

- **CSS3**
  - Flexbox
  - Variáveis CSS
  - Responsividade
  - Tipografia personalizada
  - Organização por classes

---

## Responsividade

O layout foi desenvolvido para se adaptar a diferentes tamanhos de tela, garantindo uma boa experiência tanto em desktop quanto em dispositivos móveis.

---

## Objetivos do Projeto

- Praticar HTML semântico
- Melhorar a organização e reutilização de estilos em CSS
- Trabalhar com layout responsivo
- Criar um projeto visualmente agradável sem o uso de frameworks
